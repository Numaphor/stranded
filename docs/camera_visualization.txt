Yoshi's Island Camera System Visualization
==========================================

BEFORE (Simple Following):
┌─────────────────────────────────────┐
│                                     │
│             CAMERA                  │
│               🎮                    │  Camera follows player directly
│                                     │  No lookahead, can feel reactive
│                                     │
└─────────────────────────────────────┘

AFTER (Deadzone + Lookahead):
┌─────────────────────────────────────┐
│                                     │
│        ┌─────┐                     │
│        │  🎮  │ ← Deadzone          │  Camera shows more space ahead
│        └─────┘   (24x12px)         │  Player can move in deadzone
│     ⟸────────────⟶                 │  without camera movement
│        Lookahead                    │
│        (up to 48px)                 │
│                                     │
└─────────────────────────────────────┘

MOVEMENT EXAMPLE:
Player moving right →
┌─────────────────────────────────────┐
│                                     │
│    🎮  ┌─────┐                     │  Camera shifted right to show
│        │     │ ← Deadzone          │  what's coming ahead
│        └─────┘                     │  Player appears left-of-center
│     ────────────────⟶              │  
│        Extended lookahead           │
│                                     │
└─────────────────────────────────────┘

SPECIAL EVENTS:
Trampoline jump!
┌─────────────────────────────────────┐
│                🎮                   │  Camera instantly snaps or
│                                     │  quickly moves to show landing
│                                     │  area for big vertical movements
│              ⟸─ ─ ─ ─⟶              │
│         Fast camera movement        │
│                                     │
└─────────────────────────────────────┘

Key Benefits:
✓ Smooth, natural camera movement
✓ Shows more of what's coming
✓ Prevents jittery micro-adjustments  
✓ Handles special events gracefully
✓ Respects level boundaries
